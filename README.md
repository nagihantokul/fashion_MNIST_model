# üëó Fashion MNIST Classification with Convolutional Neural Networks (CNN)

This repository contains a deep learning project that uses a Convolutional Neural Network (CNN) to classify grayscale images from the **Fashion MNIST** dataset into clothing categories (such as T-shirt, dress, sneaker, etc.).  

The project includes:

- A **Jupyter Notebook** for model training, evaluation, and experiments.
- A **Python app script** for loading the trained model and running predictions.

## üßµ Dataset: Fashion MNIST

The **Fashion MNIST** dataset is a drop-in replacement for the classic MNIST digits dataset, but with images of clothing items:

- 70,000 grayscale images
- Each image: 28 √ó 28 pixels
- 10 classes, such as:
  - T-shirt/top
  - Trouser
  - Pullover
  - Dress
  - Coat
  - Sandal
  - Shirt
  - Sneaker
  - Bag
  - Ankle boot

The dataset is commonly accessed via deep learning frameworks such as **TensorFlow / Keras**.

---

## üß† Model Overview

The core model is a **Convolutional Neural Network (CNN)**, a type of deep learning architecture especially well-suited for image classification.

Typical architecture components:

- **Input layer** for 28√ó28√ó1 grayscale images  
- **Convolutional layers** with ReLU activation  
- **MaxPooling layers** to reduce spatial dimensions  
- **Dropout layers** (optional) for regularization  
- **Fully connected (Dense) layers** for classification  
- **Softmax output layer** with 10 units (one for each class)

The model is trained to minimize **categorical cross-entropy** or **sparse categorical cross-entropy**, using an optimizer such as **Adam**.

## üñ•Ô∏è `mnistapp.py` ‚Äî Inference / Prediction Script

The `mnistapp.py` script is responsible for:

- Loading the **trained CNN model** from disk.
- Preprocessing **new input images** to match training format.
- Running the model to predict the clothing category.
- Displaying the predicted label (and possibly the probability scores).

A typical workflow inside `mnistapp.py`:

1. Load the saved model:
   ```python
   from tensorflow.keras.models import load_model

   model = load_model("fashion_mnist_cnn.h5")
   ```

2. Load and preprocess an input image:
   - Resize to 28√ó28
   - Convert to grayscale (if needed)
   - Normalize pixel values
   - Reshape to (1, 28, 28, 1)

3. Run prediction:
   ```python
   import numpy as np

   preds = model.predict(img_array)
   predicted_class = np.argmax(preds, axis=1)[0]
   ```

4. Map class index to human-readable label (e.g., "Sneaker", "Dress", etc.).
